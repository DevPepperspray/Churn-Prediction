An end-to-end machine learning project that predicts customer churn using a trained classification model, deployed as an interactive web application with Streamlit.

**Features**
Data Preprocessing: Encoding, scaling, and feature engineering
Model Training: Comparison of Logistic Regression, Random Forest, and Gradient Boosting
Interactive UI: Clean Streamlit interface for real-time predictions
Deployment Ready: Serialized model and scaler for easy deployment

**Model Details**
Input Features: Gender, Tenure, Monthly Charges, Dependents, Contract Type
Target Variable: Churn (Yes/No)
Best Model: Logistic Regression / Random Forest (specify yours)
Evaluation Metrics: Accuracy, Precision, Recall, F1-Score
